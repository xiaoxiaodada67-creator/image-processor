# 图片处理器

一个功能强大的桌面应用程序，用于批量处理图片，支持元数据移除、格式转换和图像编辑。

## 功能特性

- ✅ **批量处理** - 一键处理多张图片
- ✅ **移除元数据** - 删除 EXIF、IPTC 和 XMP 元数据
- ✅ **格式转换** - 支持 JPG、PNG、WebP、GIF 和 BMP 格式转换
- ✅ **图像编辑** - 调整亮度和对比度
- ✅ **隐私保护** - 所有处理都在本地计算机上进行
- ✅ **Windows 安装程序** - 使用 NSIS 安装程序便于安装

## 系统要求

- Windows 7 或更高版本
- 100 MB 可用磁盘空间
- 无需互联网连接

## 安装

1. 从 [发布页面](https://github.com/xiaoxiaodada67-creator/image-processor/releases) 下载最新的安装程序
2. 运行安装程序并按照屏幕上的说明进行操作
3. 从"开始"菜单或桌面快捷方式启动图片处理器

## 使用方法

### 基本工作流程

1. **选择图片** - 点击"选择图片"选择一张或多张图片文件
2. **选择输出位置** - 选择保存处理后图片的位置
3. **配置设置**:
   - 选择输出格式（JPG、PNG、WebP 等）
   - 启用/禁用元数据移除
   - 根据需要调整亮度和对比度
4. **处理** - 点击"处理所有图片"按钮，一键处理所有选定的图片

### 详细设置说明

#### 移除元数据
- **默认启用** - 移除所有 EXIF、IPTC 和 XMP 元数据
- 在分享图片时有助于保护隐私

#### 输出格式
- **JPG** - 压缩格式，适合照片
- **PNG** - 无损格式，适合图形
- **WebP** - 现代压缩格式
- **GIF** - 支持动画格式
- **BMP** - 未压缩格式

#### 亮度和对比度
- **亮度**: 调整范围 -50% 至 +50%
- **对比度**: 调整范围 -50% 至 +50%
- 使用滑块微调图像外观

## 从源代码构建

### 前置要求

- Node.js (v14 或更高版本)
- npm 或 yarn

### 安装步骤

```bash
# 安装依赖
npm install

# 启动开发模式
npm start

# 为 Windows 构建
npm run build

# 创建安装程序
npm run electron-build
```

### 开发

```bash
# 启动 Electron 和 React 开发服务器
npm start

# 应用将打开并启用热重载
```

## 输出

处理后的图片将以 `-processed` 后缀保存在选定的输出目录中。

示例：
- 输入: `photo.jpg`
- 输出: `photo-processed.jpg`

## 隐私声明

所有图像处理都在您的计算机本地进行。没有文件会上传到任何服务器。

## 故障排除

### 应用无法启动
- 尝试卸载并重新安装应用
- 确保您有管理员权限
- 检查您至少有 100 MB 的可用磁盘空间

### 处理失败
- 确保输出目录可写
- 检查您有足够的磁盘空间用于输出文件
- 先尝试用较小的图片

### 元数据仍然存在
- 某些图像格式可能不支持元数据移除
- 尝试转换为 JPG 或 PNG 格式
